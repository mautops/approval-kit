## 使用 Go Modules

```bash
go get github.com/mautops/approval-kit
```

## 版本要求

- Go 1.25.4 或更高版本

## 验证安装

```bash
go list -m github.com/mautops/approval-kit
```

## 依赖说明

Approval Kit 优先使用 Go 标准库,最小化外部依赖.当前无外部依赖.
